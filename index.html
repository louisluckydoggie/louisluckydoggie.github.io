如何安装 Python 3 的虚拟环境

Posted on 2016-10-09

因为我本机装的是 Python 2.7 ,特地写个文档来记录下如何安装 Python 3 的虚拟环境。

  1. 本机只有 Python 2 的环境，所以我们还需要去官网下载 Python 3 ，如果你是使用的 MAC ，那么安装就非常方便，从 Python 官网下载Python 3.5 的安装程序，双击运行并安装即可。

  2. MAC 下环境变量不用手动设置，安装程序已经帮你指定，所以接下来我们只要创建虚拟环境（前提是你已经安装了virtualenv）：

    $ virtualenv -p python3 learn_p3_env

    这里的-p指的是指定 Python 3 来安装虚拟环境。

  3. 接下来需要去激活虚拟环境：

    $ source ./learn_p3_env/bin/activate

    注意此时命令行会多一个learn_p3_env，接下来所有模块都只会安装到这个虚拟的环境中去。

  4. 如果你要退出虚拟环境，只要输入以下命令即可：

    $ deactivate

Linux下进程前后台切换以及管理

Posted on 2016-08-17

Linux里进程前后台相关的主要有一下几个操作:

command &   //将进程放在后台执行     
ctrl-z      //暂停当前进程 并放入后台    
jobs        //查看当前后台任务    
bg          //将任务转为后台执行    
fg          //将任务调回前台   
kill        //杀掉任务 
ps          //查看进程

如何使一个进程在后台运行

1.command &

在运行的命令后加一个&号，就会后台运行命令。

2.ctrl-z

在终端中按ctrl-z会将当前任务暂停并转入后台。

3.jobs

利用jobs命令可以查看当前后台的任务，如果加上-l参数，那么就会显示详细信息：

4.bg和fg

可以发现上图中有 Stopped 状态的进程，可以通过bg %i（i为进程的标号）命令可以将其转为运行。

如果想要将任务切换回前台，可以利用fg %i命令。

5.kill

如果想要结束任务，可以利用kill %i命令。

6.ps

ps可查看所有进程，而非当前会话。如ps -ef|grep ping可查看所有ping相关的进程：
