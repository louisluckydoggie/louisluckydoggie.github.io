如何安装 Python 3 的虚拟环境

Posted on 2016-10-09

因为我本机装的是 Python 2.7 ,特地写个文档来记录下如何安装 Python 3 的虚拟环境。

  1. 本机只有 Python 2 的环境，所以我们还需要去官网下载 Python 3 ，如果你是使用的 MAC ，那么安装就非常方便，从 Python 官网下载Python 3.5 的安装程序，双击运行并安装即可。

  2. MAC 下环境变量不用手动设置，安装程序已经帮你指定，所以接下来我们只要创建虚拟环境（前提是你已经安装了virtualenv）：

    $ virtualenv -p python3 learn_p3_env

    这里的-p指的是指定 Python 3 来安装虚拟环境。

  3. 接下来需要去激活虚拟环境：

    $ source ./learn_p3_env/bin/activate

    注意此时命令行会多一个learn_p3_env，接下来所有模块都只会安装到这个虚拟的环境中去。

  4. 如果你要退出虚拟环境，只要输入以下命令即可：

    $ deactivate
